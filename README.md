此模组为【魔女的扫帚】的非官方重制版，对代码进行了重构，并将扫帚史诗级加强（扫帚多快全靠自觉啊）。默认速度18格/秒，全维度可传，详细配置请自行设置。

目前版本：neoforge1.21.1

添加了一个魔女的扫帚物品及实体，以及魔女的帽子和衣服。

该模组以魔女之旅中伊蕾娜的扫帚为原型制作。

扫帚的操作逻辑：W/S前后、A/D转向、左alt下降 、空格上升、左ctrl加速（和疾跑一样）

右键地面即可召唤出扫帚，潜行加左键收回。\
按键盘的r键可以进行扫帚召唤，召唤范围请在配置里面修改。
